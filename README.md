## **GET STARTED**

**1. Clone it**
```bash
git clone https://github.com/YOUR_USERNAME/arcade-alley.git
cd arcade-alley
```

**2. Install it**
```bash
npm install
```

**3. Run it**
```bash
npm run dev
```

**4. Play it**

Open **[http://localhost:3000](http://localhost:3000)** and start gaming.
